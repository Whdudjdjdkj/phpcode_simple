## DİZİNİN ELEMANLARINI ÇARPMA

### DİZİNİN ELEMANLARİNİ CARPAR
  
DATA SEGMENT  
TABLE DW 1,2,6,4,5  
DATA ENDS  
  
CODE SEGMENT  
ASSUME CS:CODE, DS:DATA. 
  
START:  
  
MOV AX,DATA  
MOV DS,AX  
  
MOV AX,1  
MOV CX,5  
MOV SI,OFFSET TABLE  
  
PROGRAM:  
MOV BX,[SI]  
MUL BX  
ADD SI,2  
LOOP PROGRAM  

CODE ENDS  
END START  

OUT:240  


## XLAT KOMUTU İLE TABLO ELEMANİ COZUMLEME
